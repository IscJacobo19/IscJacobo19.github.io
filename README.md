# CV Profesional - Jose Manuel Jacobo Urzua

Sitio web personal tipo CV/portfolio para mostrar experiencia profesional, habilidades tecnicas, certificaciones y canales de contacto.

## Objetivo

Presentar un perfil profesional claro, moderno y orientado a resultados, con enfoque en desarrollo de software backend y full stack.

## Stack

- HTML5 semantico
- CSS moderno responsive
- JavaScript vanilla para interacciones ligeras
- GitHub Pages para despliegue

## Contenido

- Perfil profesional
- Experiencia y logros
- Habilidades tecnicas
- Certificaciones y educacion
- Contacto y redes
